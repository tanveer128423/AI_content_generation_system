# AI Learning Content Generator

An AI-powered platform that generates structured learning content and quizzes using Google Gemini.

## Features

- Learning Unit Creation
- AI Content Generation
- Quiz Generation
- Prompt Configuration
- Markdown Preview
- Workspace Save & Load
- BYOK (Bring Your Own Gemini API Key)

## Tech Stack

- React
- TypeScript
- Vite
- Material UI
- LangChain
- Google Gemini
